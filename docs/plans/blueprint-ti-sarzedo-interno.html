06 · Tecnologia

Stack tecnológico selecionado

Tecnologias modernas, gratuitas na fase inicial, escaláveis conforme a prefeitura cresce. Custo inicial: R$ 0/mês.

⚛️

Frontend

Next.js + Tailwind

Interface moderna, rápida e responsiva em celular. Uma base de código para os dois portais.

Gratuito
🗄️

Banco de Dados

Supabase PostgreSQL

Banco relacional gerenciado, com autenticação e segurança por linha de dados. Interface visual incluída. Uso estimado: ~54MB em 10 anos.

500MB grátis · uso real ~1%
🔐

Autenticação

Supabase Auth

Servidores: login por matrícula sem senha. Técnicos: e-mail + senha. 2FA para gestor.

Incluído
💬

WhatsApp FASE 2

Evolution API

Notificações via WhatsApp: abertura, atribuição, resolução. Implementado após validação da adoção do portal.

Opcional — Fase 2
🌐

Hospedagem

Vercel

Deploy automático a cada atualização. HTTPS incluído. CDN global. Zero manutenção de servidor.

Gratuito
📧

E-mail

Resend

Relatórios automáticos, alertas de SLA e notificações por e-mail. Uso estimado: ~500 emails/mês (17% do limite gratuito).

3k/mês grátis · uso real 17%
